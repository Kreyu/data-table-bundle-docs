# Types

The following column types are natively available in the bundle:

## Text columns

* TextColumnType
* NumberColumnType
* BooleanColumnType
* LinkColumnType

## Date and time columns

* DateColumnType
* DateTimeColumnType
* DatePeriodColumnType

## Special columns

* CollectionColumnType
* FormColumnType
* TemplateColumnType
* ActionsColumnType

## Base columns

* ColumnType
